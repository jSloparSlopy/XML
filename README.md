# Koktel Priručnik 🍸

Web aplikacija za pregled klasičnih i modernih koktela, izrađena u PHP-u s XML i JSON pohranom podataka.

## Live Demo
Aplikacija je dostupna besplatno na InfinityFree hostingu.

## Funkcionalnosti

- Pregled **100+ koktela** organiziranih u **11 kategorija** (Gin, Whiskey, Rum, Vodka, Tequila...)
- **Pretraživanje** po nazivu koktela ili sastojku u realnom vremenu
- **Filtriranje po kategoriji** s horizontalnim navigacijskim izbornikom
- **Dohvat koktela s eksternog API-ja** (`TheCocktailDB`) i spremanje u JSON datoteku
- Prikaz sastojaka s količinama po svakom koktelu
- Responzivan dizajn s karticama

## Tehnologije

| Tehnologija | Uloga |
|-------------|-------|
| PHP | Serverska logika, čitanje XML-a |
| XML | Pohrana lokalnih koktela i kategorija |
| JSON | Pohrana koktela dohvaćenih s API-ja |
| HTML/CSS/JS | Frontend, pretraživanje i filtriranje |
